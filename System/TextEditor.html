Text Editor

A text editor is different from a WYSIWYG editor because it doesn't do display formatting like bold type, or mixed font styles, sizes and colors.

Advanced text editors are typically used by computer programmers, and have text manipulation features not found or regularly used in word processors, like regular expression search and replace, clip libraries, bookmarks to tag individual lines, upfront display of line numbers, file size and the like, and the ability to save in PC, Mac and Unix formats.
