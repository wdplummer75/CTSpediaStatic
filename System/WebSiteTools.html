Web Site Tools

  •      (More options in WebSearch)
  • WebChanges: Display recent changes to the System web
  • WebIndex: List all System topics in alphabetical order. See also the faster WebTopicList
  • WebNotify: Subscribe to an e-mail alert sent when something changes in the System web
  • WebStatistics: View access statistics of the System web
  • WebPreferences: Preferences of the System web (DefaultPreferences has site-wide preferences)

  • The list above provides the SiteTools used to navigate, search and maintain a particular web. It is included within other topics using the following directive:

    %INCLUDE{"%SYSTEMWEB%.WebSiteTools"}%
