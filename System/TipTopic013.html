Preference settings

Foswiki has several levels of preferences settings. See PreferenceSettings for a full list and more information:
  1. Default settings: DefaultPreferences (never edit this topic)
    • Site name, proxy settings, access control, skin
  2. Customized site settings: SitePreferences
    • Site name, proxy settings, access control, skin
  3. Web-level settings: located in each Web, e.g. WebPreferences topic
    • Web specific access control, navigation links, colors
  4. User-level settings: In each user's home page, e.g. WikiAdmin topic
    • Settings that are only in effect when this user is logged in, such as LINKTOOLTIPINFO
  5. Topic-level settings: Affecting current topic, for web applications

Preferences can be overridden by the next level down.
