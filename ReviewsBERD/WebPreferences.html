ReviewsBERD Web Preferences

The following settings are web preferences of the ReviewsBERD web. These preferences overwrite the site-level preferences in %TWIKIWEB%.DefaultPreferences and Main.SitePreferences, and can be overwritten by user preferences (your personal topic, eg: WikiAdmin in the Main web).

Web Preferences Settings

These settings override the defaults for this web only. See full list of defaults with explanation.

  • Web Settings:
    • Set WEBBGCOLOR = #D0D0D0
    • Set SITEMAPWHAT = Review BERD Specific Materials
    • Set SITEMAPUSETO = ...collaborate on BERD reviews.
    • Set SITEMAPLIST = on
    • Default template for new topics for this web:
      • %TWIKIWEB%.WebTopicEditTemplate: Site-level default topic template

  • Set VOTEPLUGIN_DEFAULTS = open="on" global="on" bayesian="on"
