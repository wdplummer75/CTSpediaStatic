Contributing Graph Classes Using Templates

IF you would like to provide a class graphic:
  • Add a WikiWord for your class graphic in the box below and click on "Create by name."
  • The template has fields, which you are free to use or to delete.
  • Java script for the coding was added for coding. It will be formatted correctly if you add add your code between the 
  •  %CODE{lang="java"}% and %ENDCODE%
    .
