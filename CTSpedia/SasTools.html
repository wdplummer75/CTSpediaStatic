SAS Macros

These are components that can be used within your own SAS programs. Headings at the top of each file describe how to call or use them, and examples are provided. See below for guidelines on calling these from within your programs.

  • Logistic Regression

  • Categorize Groups variables into discrete, continous and Dates
  • Pearson and Spearman Correlations
  • Partial Least Squares
  • CrossPlots
  • Boxplots several variables side-by-side
  • Creates Standand Error plots for a list of variables
  • False Discovery Rate
  • Least Squares Mean Plot for Mixed Models
  • Least Squares Mean Plot for GEE Models
  • Area under curve calculation for GEE Models
  • Finding cutpoint for a continous predictor using ROC
  • Power and/or effect size difference calculation (longitudinal data)
  • Power and/or effect size difference calculation for group study (longitudinal data)
  • Zero-inflated Poisson Model Check and Fit
  • Brief TTest
  • Match Sample
  • Utilities
