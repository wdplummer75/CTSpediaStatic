Notification of Changes

Return to Help for Contributors

If you want to be notified of changes on the CTSpedia web,

  • Go to this page, click on edit, and add your email address.

If you only want to be notified of changes on certain pages,

  • Put a colon after your address
  • Add the wiki name of the page that you wished to be notified about
  • If there are a number of pages stemming from your original page, mark the number of stems that you wish to be notified about.

Notifications

  • Occur nightly at midnight when there are changes to the specify pages.
  • You can always click on Recent Changes on the right menu to see the work people are doing.
