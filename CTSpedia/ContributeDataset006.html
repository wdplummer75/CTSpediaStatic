Description of Source Data published by Nawata et al (2004)(on course CD). The data were taken from the authors\x92 Figure 1, a scatterplot, and so only approximate the actual values used by the authors.
Description of Dataset
Codebook File rmr.dta Codebook
group urinary excretion of albumin group (U-Alb)
a = U-Alb < 30 mg/d
b = 30 mg/d ≤ U-Alb ≤ 300 mg/d
c = 300 mg/d < U-Alb
lbm lean body mass (kg)
rmr resting metabolic rate (kJ/h/m2)
