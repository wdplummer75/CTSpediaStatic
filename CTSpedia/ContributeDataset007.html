Dataset Title Smoking Cessation Study (smoke.csv)
Contributor/Email GregoryStoddard
CTSA Utah
Acknowledgment Please cite the appropriate contributors/authors/contacts when using or adapting these materials.
Description of Source This dataset was distributed with the Rosner (1995) biostatistics textbook. In this dataset, 234 smokers who expressed a willingness to quit smoking were followed for one year to estimate the proportion of recidivism (quit for a time and then started again).

Description of Dataset
Codebook Variable Labels

id subject identification number
age age quit smoking
gender gender
1=male, 2=female
cigs smoking habit at time quit smoking (cigarettes/day)
days days abstinent (up to 365, which ends the follow-up period)

Keywords
See Also Appendix 1 - Stoddard GJ. Biostatistics and Epidemiology Using Stata [unpublished manuscript] University of Utah School of Medicine, 2009
