To add a new point in a Controversy page:

1. Click the Edit button on this page.
2. Copy everything from "Heading text here" to the bottom of the page.
3. Click the red "Cancel" button.
4. Return to the Controversy page you wish to edit.
5. Click the edit button on that page.
6. Scroll to where you want to add the new point and position the cursor there.
7. Paste what you copied at step 2.
8. Edit the heading, introduction text, and column headings.
9. Click the Save button.
