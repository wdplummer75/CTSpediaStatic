Type of Tool SAS Macro
Title Least Squares Mean Plot GEE
Programmer/Email YinglinXia
Contributing Site Rochester
Description LSM Plot GEE macro computes and plots the least squares means from SAS PROC GENMOD procedure to implement the Generalized Estimating Equations (GEE) comparing across treatment and control groups over time. The user can choose one of the available working correlation matrices in PROC GEENMOD. The response variable can be continuous, binary, but not ordinal since least squares means are not available in current SAS version (9.2) for ordinal outcome.

Macro Parameters The macro parameters are as follows:
DSname =name of SAS data set
DVar =name of response variable
time =name of time variable
group =name of treatment groups, which are dichotomized as treatment and control
Endtime =name of indicator of end time point variable
CovCat =name of categorical covariates
CovCon =name of continuous covariates
id =name of the variable for subject id
type =type of working correlation matrix
format =name of output format
title =name of title will be specified
