Title Standard Error Plot
Programmer/Email ArthurWatts
Contributing Site Rochester
Description This set of Macros creates StdErr plots for several variables over time and staggers the StdErr bars by an optional grouping variable. This procedure can also be used after calling the Deltas macro, to see the distribution of changes from baseline.
