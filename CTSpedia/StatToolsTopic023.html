Title Matched Sample
Programmer/Email KimberlyKaukeinen
Contributing Site Rochester
Description This is a macro that will match 2 groups of observations by X number of variables. This macro will input a file, process, and output a file with only the matched observations on it. The input file needs to have a variable on it that identifies the groups with a 0 or 1. There needs to be a variable on the file that identifies the patient. In the calling program, a libname SAVE needs to be set up for the macro to save the output file to. The output file will be called matched_sample.

Macro Parameters filein: input file name
group: variable that identifies the 2 groups; has a value of either 0 or 1
ptid: variable that identifies the patient id
matchby: list of variable names that the sample will be matched on
last: the last variable name in matchby above


An Example:
filein: filein
group: group
ptid: ptid
matchby: gender age_grp duration
last: duration
