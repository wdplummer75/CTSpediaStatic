SitePreferences

Foswiki's default preferences are located in System.DefaultPreferences, however settings defined here have override priority and should be used for local customisations.

  • Appearance
  • Finalisation

%EDITPREFERENCES{"System.DefaultPreferencesForm"}%

Appearance

Customize logo and tooltip texts:

  • Set WIKILOGOIMG =
  • Set WIKILOGOALT = CTSpedia
  • Set WIKILOGOURL = http://biostat1478.dhcp.mc.vanderbilt.edu/foswiki/bin/view/Main/WebHome

Give your site a specific name:

  • Set WIKITOOLNAME = Foswiki

Customize your favicon:

  • Set FAVICON = http://biostat1478.dhcp.mc.vanderbilt.edu/foswiki/pub/Main/SitePreferences/favicon.ico

Customize your top bar background image and color
  • Set WEBHEADERART = /foswiki/pub/System/PatternSkin/header5.gif
  • Set WEBHEADERBGCOLOR = #ffffff

Finalisation

FINALPREFERENCES locks site-level preferences that are not allowed to be overridden by WebPreferences or user preferences:

  • Set FINALPREFERENCES = ATTACHFILESIZELIMIT, PREVIEWBGIMAGE, WIKITOOLNAME, WIKIHOMEURL, ALLOWROOTCHANGE, DENYROOTCHANGE, DOCWEB, WIKIWEBMASTER, WIKIWEBMASTERNAME, WIKIAGENTEMAIL, WIKIAGENTNAME

  • favicon.ico: favicon.ico
